# GitHub Repository Topics Setup

To maximize discoverability of the Gravity-Lang project on GitHub, add the following topics to your repository:

## How to Add Topics

1. Go to: https://github.com/dill-lk/Gravity-Lang
2. Click the ⚙️ gear icon next to "About" (top right)
3. In the "Topics" field, add the topics listed below
4. Click "Save changes"
